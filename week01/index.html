Some Syntax for localStorage

  • localStorage.getItem
  • localStorage.setItem

Click here for an example of localStorage

To store an item make sure to use the key, value pair (i.e. localStorage.setItem(storyName,storyHTML)).

To retrieve from localStorage you only need to use the key (i.e. localStorage.getItem(storyName)).

Mobile Design Notes

Make your app:

  • Frictionless
  • Reachablility Matters (how you hold the device)
  • Speed matters - slow means you loose traffic
    • use svg and webfonts
    • Minify css and js code
    • gzip files

Menu and navigation

  • move the menu to a reachable place (bottom)

Designing Mobile Forms

  • Ditch the Labels - diplay inside field
  • Use float labels
  • get rid of as many feilds as you can
  • enable autocomplete on form
  • show password - allow to hide
  • bring up the correct keyboard - for a credit card number bring up the keypad
  • link telephone numbers

Mobile Advertising

  • Keep it simple and sleek
  • Don't tick off your users
  • Give your users content not more advertising
